# cipher-lab
Text encryption tool. Definitely not secure enough to protect state secrets or your conversations with your ex.

## About

CipherLab is a command-line application in C that demonstrates and expands on the classical Caesar cipher. The project is a part of programming assignment. 

## Features

1. Encrypt text
2. Decrypt text
3. Input validation
4. Interactive menu
5. Built-in explanation of the Caesar cipher
6. Encryption statistics
7. Preserves punctuation, numbers and spaces

## Requirements

1. CS50 Library (`cs50.h`)
2. GCC / Clang
3. CS50.dev (recommended)


## Compile

make cipher-lab


## Run

./cipher-lab


## Disclaimer

This project is intended for educational purposes only. It is not secure encryption.
